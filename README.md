# TradeGold Project Overview

This project focuses on utilizing machine learning models for trading prediction based on candlestick analysis and integrating Level 2 data support for enhanced decision-making.

## Updated Steps:
1. **Machine Learning Models**: Developing and testing various algorithms to predict market trends.
2. **Candlestick Analysis**: Implementing techniques to analyze price movements and identify potential trading opportunities.
3. **Level 2 Data Support**: Enhancing trading decisions by incorporating more granular market data to identify support and resistance levels.
